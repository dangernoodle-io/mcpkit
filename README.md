# mcpkit

[![Go](https://img.shields.io/badge/Go-1.26-00ADD8?logo=go)](https://go.dev/)
[![Build](https://github.com/dangernoodle-io/mcpkit/actions/workflows/build.yml/badge.svg)](https://github.com/dangernoodle-io/mcpkit/actions/workflows/build.yml)
[![Coverage Status](https://coveralls.io/repos/github/dangernoodle-io/mcpkit/badge.svg?branch=main)](https://coveralls.io/github/dangernoodle-io/mcpkit?branch=main)

mcpkit is a host-agnostic Go framework for building MCP servers, with compile-time
capability composition over the official
[`modelcontextprotocol/go-sdk`](https://github.com/modelcontextprotocol/go-sdk).

**Status: alpha.** The API is still taking shape; expect breaking changes between phases.

See the [wiki](https://github.com/dangernoodle-io/mcpkit/wiki) and
`handoffs/mcp-framework-design.md` for the design rationale and roadmap.
